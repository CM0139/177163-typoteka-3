# Переменные среды

## Общее

`NODE_ENV` — окружение (production/development);
`API_PORT` — порт api сервера;  
`FRONT_PORT` — порт сервера приложения;  

## СУБД

`DB_NAME` — название базы данных;  
`DB_USER` — имя пользователя;  
`DB_PASSWORD` — пароль пользователя;  
`DB_HOST` — хост;  
`DB_PORT` — порт;  
`DB_TYPE` — диалект;
